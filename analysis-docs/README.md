# 📊 Huobao Drama 项目分析文档

## 🎬 项目简介

**火宝短剧（Huobao Drama）** 是一个基于 AI 的短剧自动化生产平台，实现了从剧本生成、角色设计、分镜制作到视频合成的完整工作流。

- **技术栈**: Go 1.23 + Vue 3 + Gin + SQLite + FFmpeg
- **架构**: DDD（领域驱动设计）分层架构
- **核心功能**: AI剧本解析、角色设计、分镜生成、视频合成
